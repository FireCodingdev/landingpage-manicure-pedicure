# Studio Nails – Modelo Landing Page Premium Estética
Esta é uma Landing Page (modelo) de Alta Conversão desenvolvida para profissionais de Nail Design, Manicure e Pedicure que buscam uma presença digital sofisticada e eficiente. O projeto foi estruturado com foco em experiência do usuário (UX), carregamento rápido e prontidão para campanhas de tráfego pago.

🚀 Visualização em Tempo Real
O projeto está publicado e pode ser acessado em:

👉 https://firecodingdev.github.io/landingpage-manicure-pedicure/

✨ Funcionalidades Principais
Design Responsivo Premium: Interface elegante adaptada para smartphones, tablets e desktops.

Foco em Conversão: Botões de Call-to-Action (CTA) estrategicamente posicionados para agendamento.

Integração com WhatsApp: Formulário inteligente que valida os dados e redireciona para o WhatsApp com mensagem pré-preenchida.

Marketing Ready (Trackeamento): Estrutura pronta para instalação de Google Tag Manager, Google Ads e Meta Pixel.

Seções Estratégicas: * Exposição de serviços com valores.

Prova social (depoimentos de clientes).

Mapa de localização integrado.

Política de Privacidade integrada (Conformidade com LGPD).

🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando tecnologias puras (Vanilla) para garantir o máximo de performance e facilidade de manutenção:

HTML5: Estruturação semântica.

CSS3: Estilização personalizada com variáveis modernas e animações de revelação ao scroll.

JavaScript (ES6+): Lógica de formulário, máscaras de entrada e efeitos de interface.

GitHub Pages: Hospedagem e deploy automatizado.

📈 Preparado para Tráfego Pago
Diferente de sites institucionais comuns, este código já possui os "hooks" para monitoramento de eventos:

Evento de Conversão: Disparo de gtag_report_conversion ao clicar nos botões de agendamento.

Lead Tracking: Disparo de evento Lead do Facebook Pixel após o envio do formulário.

🔧 Como usar este projeto
Clone o repositório:

Bash
git clone https://github.com/FireCodingdev/landingpage-manicure-pedicure.git
Personalize as informações:

Edite o arquivo index.html para alterar o número do WhatsApp, os nomes dos serviços e os preços.

Substitua os IDs de rastreamento (GTM-XXXXXXX, AW-XXXXXXX, etc.) pelos seus IDs reais.

Suba para o seu GitHub: O deploy será automático via GitHub Pages.
